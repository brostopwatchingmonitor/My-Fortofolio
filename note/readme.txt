untuk semantic harap gunakan
- <main> untuk bacaan utama
- <nav> untuk membuat setiap ada kalimat link di luar bacaan utama
- <section> untuk tanda beberapa tambahan utama / untuk membungkus tanpa perubahan
- <mark> untuk highlight tulisan
- <article> untuk membungkus berita di web
- <aside> untuk catatan kecil di beberapa bagian
- <footer> untuk bagian bawah web
